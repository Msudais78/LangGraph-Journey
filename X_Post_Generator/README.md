# Iterative X Post Generator

This is a LangGraph-based workflow for generating high-quality X (Twitter) posts through an automated cycle of generation, evaluation, and optimization. It uses an LLM to generate an initial draft based on a topic, another LLM to score the draft against 7 rigorous criteria, and a third LLM to iteratively improve the post until it passes the quality threshold.

## Contents

- `generate_notebook.py`: Python script containing the source code to build the complete Jupyter Notebook (`x_post_generator.ipynb`) programmatically.
- `x_post_generator.ipynb`: The self-contained notebook containing the LangGraph code, markdown explanations, and execution cells, generated by `generate_notebook.py`.

## Features

- **Generator (Google Gemini 2.5 Flash)**: Initial X post drafting.
- **Evaluator (Groq llama-3.3-70b-versatile)**: Assigns per-criterion scores from 1-10 on dimensions like "Hook Power", "Platform Fit", "Topic Authority", and "Originality".
- **Optimizer (Groq llama-3.1-8b-instant)**: Rapid text refactoring based on evaluator scores emphasizing the weakest areas.
- **Automated Routing**: Code iteratively passes the text between Generator, Evaluator, and Optimizer until an average score threshold is passed (default is 9.0) or `MAX_ITERATIONS` is reached.
- **Mermaid Graph Visualization**: Renders an execution tree of the compiled `StateGraph` nodes in the Notebook block context using `IPython.display.Image` and `.draw_mermaid_png()`.

## Quick Start

1. Ensure your environment dependencies are installed: `pip install -r requirements.txt`.
2. Ensure you have a `.env` file containing your valid `GOOGLE_API_KEY` and `GROQ_API_KEY`.
3. You can either execute `generate_notebook.py` directly to overwrite `x_post_generator.ipynb`, or simply explore and run the cells within `x_post_generator.ipynb` directly.
