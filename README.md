## 🚀 How to Use This Repo Locally

If you'd like to run any of these projects on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Msudais78/LangGraph-Journey.git](https://github.com/Msudais78/LangGraph-Journey.git)
   cd LangGraph-Journey

```

2. **Set up a virtual environment (Recommended):**
```bash
python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`

```


3. **Install dependencies:**
Navigate into the specific project folder and install its requirements:
```bash
cd project-01-basic-chatbot
pip install -r requirements.txt

```


4. **Environment Variables:**
Copy the `.env.example` file in the project folder to a new file named `.env` and add your API keys. **Never commit your `.env` file to GitHub.**
